# Compatibility Test Cases

## Overview

This folder includes test cases aimed at validating the application’s performance across various browsers, and screen sizes.

All test cases are written for demonstration purposes and may differ from real project standards.

---

## Structure

The folder includes the following files:

- [Browsers](Browsers.md) – contains cross-browser compatibility test cases  
- [Screen Resolutions](Screen_Resolutions.md) – contains test cases for screen size compatibility
