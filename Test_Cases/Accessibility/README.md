# WCAG Checks – Accessibility Test Suite

This document contains test cases designed to verify compliance with WCAG accessibility guidelines. The focus is on ensuring that users with disabilities can effectively interact with the application, including support for assistive technologies and adaptive user behaviors.

---

> **Note**: These test cases are provided as sample scenarios. Actual implementation, structure, naming conventions, and required fields may vary depending on project standards, QA processes, and team practices. This suite demonstrates one possible approach to accessibility validation.

---

## Structure  

- [WCAG.md](./WCAG.md) - test cases based on WCAG 2.1 success criteria

---

## Scope

Test cases cover:

- Audio playback control (pause/stop functionality)
- Video accessibility (captions and synchronization)
- Time-based interactions and session limits
- Screen orientation adaptability (portrait/landscape)
- Text scalability and zoom support
- Form validation and error identification
